## How to Find Emails

### Method 1: YC Company Directory
1. Go to https://www.ycombinator.com/companies
2. Search company name
3. Find founder LinkedIn
4. Use Hunter.io or RocketReach

### Method 2: Company Website
- Check "About" / "Team" page
- Often has founder emails or contact form

### Method 3: LinkedIn
- Find founder profile
- Send InMail or connection request
- Message with pitch

### Method 4: Hunter.io
```
Input: company domain (e.g., univerbal.app)
Output: Common email patterns + verified addresses
Cost: FREE for 25 searches/month
```
