## üí° Pro Tips

### Batch Ordering

Order **3 variations** from same creator:
- Hook variation: "Wait" vs "Question" vs "Bold Claim"
- Emotion variation: Excitement vs Curiosity vs FOMO
- CTA variation: Direct vs Urgency vs Scarcity

**Why?** Find winning pattern faster + volume discount

### Rapid Testing Strategy

```
Week 1: Order 10 videos ($1,000-1,500)
Week 2: Analyze with Creative API
Week 3: Test top 5 on Spark Ads ($500 budget)
Week 4: Scale winner to $5,000+
```

### Pattern Insights

After testing 20+ creatives, check top patterns:

```bash
curl http://localhost:8000/api/v1/creative/patterns/top?product_category=lootbox&metric=cvr
```

**Use insights to refine future briefs!**

Example: If "wait" hook + "excitement" emotion = 15% CVR, order MORE videos with that pattern.
